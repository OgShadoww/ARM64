# Registers

## Basic registers
---
```c
x0-x30 // Basic 64 bit registers
w0-w30 // The half of xN register first 32 bits registers
sp // stack pointer
pc // program counter
```

## Registers with special meaning:
---
```c
x0-x5 // Function arguments
x16   // Syscall number MacOS
x8    // Syscall number Linux

```
